# Kalvium Assignment Video Generator

A simple automation tool that generates explainer videos for Kalvium assignments.

## Features

- Generate explanation script from assignment content
- Convert script to voiceover using TTS
- Generate MP4 video automatically
- Simple React UI
- Download-ready video output

## Tech Stack

Frontend
- React
- Vite

Backend
- Node.js
- Express
- FFmpeg
- Piper TTS

## Project Structure

/client → React frontend  
/server → Express backend

## Installation

### Install client

cd client  
npm install

### Install server

cd server  
npm install

## Run Project

Start backend

cd server  
npm run dev

Start frontend

cd client  
npm run dev
